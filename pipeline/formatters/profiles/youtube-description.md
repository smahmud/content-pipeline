## Description Guidelines

Create an optimized YouTube description:

1. **First 150 Characters**: Critical for search - include main keyword and hook
2. **Above the Fold** (first 2-3 lines visible):
   - Compelling summary of video value
   - Include primary keywords naturally
   - Add a CTA (subscribe, like, comment)

3. **Timestamps/Chapters**:
   - Format: 0:00 - Introduction
   - Include all major sections
   - Use descriptive chapter titles

4. **Body Content**:
   - Expand on video topics
   - Include relevant links
   - Add social media links
   - Mention related videos

5. **SEO Elements**:
   - Natural keyword placement
   - Hashtags (3-5 relevant ones)
   - Links to resources mentioned

6. **Footer**:
   - Subscribe reminder
   - Social links
   - Contact/business inquiries

Keep the tone matching the video style while maximizing searchability.
